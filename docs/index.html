llm-recall

跨厂商 LLM CLI 会话搜索 + 恢复 — fzf 风格，单 Go 二进制

# macOS / Linux
brew install xiao98/tap/llm-recall

# Windows
scoop bucket add xiao98 https://github.com/xiao98/scoop-bucket
scoop install llm-recall

它解决什么

  • 同时在用 Claude Code / Codex / Gemini？三家会话散落在三处目录、三套 CLI、三个 --resume 语义。
  • 统一 TUI：输入即筛，多关键词 AND，中文按 unicode 字处理。
  • 回车直接 resume：自动 cd 到原 cwd，调对应 CLI（Gemini 退化交互式 /chat resume）。
  • 终端原生 stats / gold / card：lipgloss 渲染，截屏即传播，无 PNG 后端。
  • BYOK：gold / card 调你自己的 LLM key，对话内容不经任何中转。

30 秒上手

llm-recall                    # TUI 模糊搜索（默认 dry-run）
llm-recall stats              # GitHub 风格贡献日历 + 4×2 stats 面板
llm-recall gold --days 7      # LLM 抽你说过的 Top 10 金句
llm-recall card <session-id>  # 单会话 lipgloss 圆角名片

其他安装方式

# Go install (任何平台)
go install github.com/xiao98/llm-recall/cmd/llm-recall@latest

# From source
git clone https://github.com/xiao98/llm-recall && cd llm-recall
go build -o llm-recall ./cmd/llm-recall

隐私

  • 本地 SQLite 索引，不上传任何对话内容。
  • stats 纯本地聚合，无网络出口。
  • gold / card BYOK：调你自己配的 LLM endpoint，调用前 5 类 PII 正则脱敏 + cost confirm + 7 天缓存。
  • 无 telemetry，无 crash report，无"匿名统计"——一行也没有。
  • 首次启动 onboarding 一次同意流；--no-promo 关全部 sponsored 痕迹。
